WeatherStatistics
=================

Цель WeatherStatistics: дать пользователю:
- среднестатистические показания погоды в г.Москве на выбранную дату в течение года (температура и давление воздуха, скорость ветра, наличие облаков и осадков),
- рекомендации по предметам одежды и активностям в течение дня,
- сравнение с прогнозом Гидрометцентра России (на ближайшую дату). 

Переменный сарказм по поводу московской погоды не является случайным в данном проекте.

Установка
---------

Создайте виртуальное окружение и активируйте его:

```
    На Windows: env\Scripts\activate
```
```
    На Mac или Linux: source env/bin/activate
```
После этого в виртуальном окружении выполните:

```
    pip install -r requirements.txt
```
